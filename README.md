Narratree is a node-based narrative design tool inspired by Articy Draft for branching stories, dialogue, and game logic. It is a single-file, browser-based editor that runs locally with no build step.

## Features
- Drag-and-drop node graph with branching connections
- Pan/zoom canvas with fit-to-view
- Node types, colors, and inline dialogue editing
- Connection conditions, variable updates on traverse, and on-enter actions
- Sidebar project structure for graphs, characters, items, locations, variables, and notes
- Local autosave plus JSON import/export

## Getting started
1. Open `index.html` in a modern browser (Chrome, Edge, or Firefox).
2. Double-click empty space to add a node, or use the "+ Node" button.
3. Connect nodes by clicking connection points.
4. Save your project with `Ctrl+S` or the Save button (JSON download).

## Controls
- Double-click empty space: add node
- Drag empty space: pan
- Mouse wheel: zoom
- F: fit to view
- Click connection point: connect
- Right-click connection point: delete a connection
- Drag node header: move node
- Click text: edit
- Ctrl+S: save
